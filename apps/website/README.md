# Promptyard website

This directory contains the code related to the promptyard website. Please follow the instructions in the README file to
get started.

## Getting started

- Make sure the database server and keycloak server are up and running by running `docker compose up -d` from the
  repository root. Check with `docker ps` if all health checks are green.
- `pnpm dev` - Runs the server in dev-mode.
